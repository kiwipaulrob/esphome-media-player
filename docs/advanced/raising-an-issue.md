# Raising an Issue

If you've gone through [Troubleshooting](/advanced/troubleshooting) and still need help, open an issue on GitHub.

## Before you open an issue

1. **Search existing issues** — your problem may already be reported or resolved: [GitHub Issues](https://github.com/jtenniswood/esphome-media-player/issues)
2. **Check you're on the latest firmware** — re-flash using the [web installer](/installation) to make sure you have the latest version.

## What to include

To help diagnose the problem quickly, include the following:

- **Device** — which hardware you're using (e.g. ESP32-S3 4848S040 4", ESP32-P4 86 Panel 4", ESP32-P4 JC1060P470 7")
- **Installation method** — web installer or manual ESPHome setup
- **Description** — what you expected vs. what happened
- **Steps to reproduce** — how to trigger the issue
- **Logs** — ESPHome device logs from the dashboard or Home Assistant (go to **Settings → Devices & Services → ESPHome** → your device → **Logs**)
- **Screenshots** — photos of the screen if it's a display issue

## Open an issue

[Open a new issue on GitHub](https://github.com/jtenniswood/esphome-media-player/issues/new)
