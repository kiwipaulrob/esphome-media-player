# Guition ESP32-P4 JC8012P4A1 (10.1")

10.1" (1280×800) touchscreen panel with ESP32-P4 and ESP32-C6 coprocessor for WiFi/Bluetooth. Uses a side-panel layout: album art on the left, track info on the right. Tested with Google and Sonos speakers via Home Assistant.

![Guition ESP32-P4 JC8012P4A1 showing Home Assistant media artwork](../images/guition-esp32-p4-jc8012p4a1-example1.jpg)

## Install

Click **Install** to flash the firmware directly from your browser. No ESPHome dashboard or YAML required.

<InstallButton device="jc8012p4a1" />

::: tip
If you prefer to install via the ESPHome dashboard, see [ESPHome Config](/advanced/esphome-config) (this device uses the `devices/guition-esp32-p4-jc8012p4a1` package).
:::

## Rotation

The device supports **Screen Rotation** values of `0`, `90`, `180`, and `270`. `90` is the default landscape view. `0` and `180` switch to a portrait layout with album art at the top and track details underneath.
